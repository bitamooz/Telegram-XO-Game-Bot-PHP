# ربات بازی XO (دوز) تلگرام - نسخه ۱

این ربات بازی XO دو نفره با عضویت اجباری کانال طراحی شده و کاملاً رایگان است. کاربران می‌توانند با دوستان خود بازی کنند و فقط باید در کانال عضو باشند.

## ویژگی‌های این ربات:
- **بازی دو نفره** با کیبورد تعاملی  
- **عضویت اجباری** در کانال  
- **تشخیص برنده** و مساوی  
- **ری‌استارت** و **ترک** بازی  
- **کاملاً فارسی** و با ایموجی  
- **راه‌اندازی سریع** در کمتر از ۵ دقیقه!

---

## نصب و راه‌اندازی

1. **دانلود سورس**  
   ابتدا سورس را از GitHub دانلود کنید.

2. **تنظیمات اولیه**  
   - فایل `index.php` را باز کنید.  
   - **خط ۱۸**: توکن ربات خودتان را وارد کنید.  
   - **خط ۲۰**: آیدی کانال عضویت اجباری (بدون `@`) را قرار دهید.  

3. **آپلود روی هاست**  
   فایل‌ها را روی هاست یا سرور خود قرار دهید.

4. **ست کردن وب‌هوک**  
   کافی است از لینک زیر استفاده کنید تا ربات به سورس شما متصل شود:
   ```bash
   https://api.telegram.org/bot[TOKEN]/setWebhook?url=[YOUR_DOMAIN]
   
فقط `TOKEN` و `YOUR_URL` را با اطلاعات خودتان جایگزین کنید!

تمام شد! حالا می‌توانید از ربات بازی XO استفاده کنید و کاربران خود را سرگرم کنید!
